ShellPay is a simple way to integrate TurtleCoin payments on ecommerce stores!

Getting Started

The first step to integrate shellpay is to generate a key pair. This will generate keys that will be used to authenticate API calls. Save these keys in a secure location on your server.

Next, create a http handler that will listen for payment notifications from ShellPay. Make sure you authenticate the HMAC signature of the raw request data. This is to verify that the incoming request was delivered by ShellPay.

See the API Docs.
